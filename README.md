# PiLess

Этот репозиторий содержит установочный скрипт `install.sh`, который разворачивает на Raspberry Pi Wi‑Fi точку доступа и проксирует весь трафик через VLESS+reality (Xray). Пользователь передаёт одну ссылку VLESS, скрипт парсит её, устанавливает Xray, настраивает конфигурацию и запускает сервисы.

### Требования

* Raspberry Pi под Linux с systemd
* Подключение к интернету
* Запуск от root

### Быстрый запуск

```
sudo bash install.sh
```

Во время выполнения скрипт запросит ссылку вида `vless://...`.

### Пример ссылки

```
vless://545460fd-452a-4dd4-bf50-cfb44f310419@134.234.123.123:443?type=tcp&encryption=none&security=reality&pbk=zGzsg9DqEwGuffsduWHmx8irzYPE9o2U6ukj0VXzAQ&fp=chrome&sni=google.com&sid=06d996&spx=%2F#Dark-tch2m0m2
```

### Что делает скрипт

1. Парсит VLESS ссылку
2. Устанавливает зависимости через пакетный менеджер
3. Скачивает и устанавливает Xray
4. Создаёт конфигурацию Xray с transport reality
5. Создаёт и запускает systemd сервис
6. Включает Wi‑Fi hotspot через NetworkManager
7. Настраивает redsocks и iptables для перенаправления трафика клиентов через SOCKS

### Примечания по безопасности

* Скрипт не сохраняет ссылку в открытом виде в публичных местах
* Рекомендуется проверять содержимое перед запуском
